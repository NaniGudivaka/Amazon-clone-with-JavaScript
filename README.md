# Amazon Clone

A responsive e-commerce web application built using HTML, CSS, and JavaScript.

## Features

* Displays products dynamically from a data file.
* Easily add new products by updating the product data.
* Add products to the shopping cart.
* Update product quantities in the cart.
* Calculate order totals and shipping costs.
* Checkout page with order summary functionality.
* Orders page to view placed orders.
* Order tracking page to track delivery status.
* Responsive user interface inspired by Amazon.

## Technologies Used

* HTML
* CSS
* JavaScript (ES6)
* Modular JavaScript Architecture

## Project Structure

* `data/` – Product and cart data
* `images/` – Product and UI images
* `scripts/` – Application logic
* `styles/` – Styling files
* `utils/` – Utility functions
